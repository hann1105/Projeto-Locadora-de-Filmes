Esse projeto foi feito com o propósito de atender o desafio dado pelo professor na cadeira de Estruturas de Dados, nele foram usadas listas encadeadas, Fila e Pilha como base para o projeto. 
  As listas encadeadas foram usadas para o sistema priincipal da locadora de filmes, com métodos para inserir e retirar filmes, buscar filmes por Nome e Gênero, Ordenação com Insertion Short 
com base na ordem alfabética dos filmes. Também foram utilizadas uma fila com lista encadeada para a fila de Reservas dos filmes, em que cada filme tem sua prórpia fila de reserva, assim 
quando um filme já estiver alugado, o outro cliente que deseja ele entrará para fila, dando assim o conceito de First IN First Out, também utilizei uma pilha para mostrar os filmes 5 filmes
que foram alugados mais recentemente para ter esse controle no sistema da Locadora. 
  O sistema é composto pela parte do funcionário e do cliente, inicialmente o cliente criará seu login por meio do CPF para ter acesso ao menu do cliente que terá todas as opções disponíveis,
como alugar, devolver, buscar filme, ver os filmes de cada gênero e assim por diante se o CPF for o do funcionário, abrirá o menu do sistema, que terá as opções para o funcionário ter controle
e entender o sistema da locadora, assim como adicionar novos filmes, retirar e ordenar.
